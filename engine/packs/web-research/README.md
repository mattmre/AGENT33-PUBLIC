# web-research

Web search and research workflow pack for AGENT-33.

## Skills

- **search-web**: Search the web and return ranked, cited results
- **summarize-results**: Synthesize multiple sources into a coherent summary
- **extract-facts**: Extract discrete, verifiable facts with citations

## Usage

```bash
agent33 packs validate engine/packs/web-research
agent33 packs apply web-research
```

## Tool Requirements

- `web_fetch` must be enabled in the agent's tool configuration
