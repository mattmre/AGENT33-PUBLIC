# meeting-notes

Meeting transcript processing pack for AGENT-33.

## Skills

- **extract-action-items**: Extract action items with owners and deadlines
- **summarize-transcript**: Produce structured meeting notes from a transcript
- **identify-decisions**: Record key decisions with rationale and attribution

## Usage

```bash
agent33 packs validate engine/packs/meeting-notes
agent33 packs apply meeting-notes
```

## Tool Requirements

No additional tools required beyond defaults.
