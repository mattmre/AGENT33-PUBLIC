# Code of Conduct

This project follows the
[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Report concerns to the maintainers via GitHub at
[mattmre/AGENT33-PUBLIC](https://github.com/mattmre/AGENT33-PUBLIC/issues).

[GitHub's private vulnerability reporting](https://github.com/mattmre/AGENT33-PUBLIC/security/advisories/new).
